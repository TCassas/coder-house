# Tengu Store

Final project for React course by CoderHouse

## Idea
Manga e-commerce

## Run it
Clone, cd, install, start, that's it.
```
git clone https://github.com/TCassas/coder-house.git
cd coder-house
npm install
npm start
```
This will boot up the react application on port 3000.
